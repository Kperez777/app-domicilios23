# app-domicilios23
App de Domicilios en C

Aplicación de consola desarrollada en lenguaje C que simula un sistema básico de pedidos a domicilio.

Funcionalidades
Ver menú de productos
Agregar productos al carrito
Ver carrito y total
Eliminar productos del carrito
Confirmar pedido
Vaciar carrito
Productos

El sistema incluye un catálogo con:

Hamburguesa
Pizza
Perro caliente
Papas fritas
Gaseosa
Helado
Cómo ejecutar
Compilar el programa:
gcc main.c -o app
Ejecutar:
./app
Tecnologías usadas
Lenguaje C
Consola (CLI)
